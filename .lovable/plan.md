

# Neel Joshi — Portfolio Website

## Overview
A modern, creative, single-page portfolio website for Neel Joshi, Full Stack Developer. Features light/dark theme toggle with a blue, white, blue-gray, and black color palette. Inspired by the bold, editorial design style shown in the reference image.

---

## Sections

### 1. Navigation
- Fixed top navbar with smooth scroll links: Home, About, Skills, Experience, Projects, Services, Contact
- Theme toggle button (sun/moon icon)
- Mobile hamburger menu

### 2. Hero Section
- Full-width section with animated gradient background (blue/gray tones)
- Large bold name "Neel Joshi" in editorial typography style (inspired by reference)
- Title: "Full Stack Developer"
- Tagline: "Building modern, scalable, and user-centric web applications."
- Placeholder profile picture area with clean styling
- CTA buttons: "View Portfolio" and "Contact Me"
- Subtle decorative elements (dots, lines, stars) like the reference

### 3. About Me
- Personal bio highlighting Full Stack skills
- Timeline-style education section with icons:
  - BCA (2017–2020)
  - MCA (2020–2022)
- Clean two-column layout

### 4. Skills & Expertise
- Categorized skill cards with icons:
  - **Frontend:** HTML, CSS, JavaScript, Vue.js, React.js, Bootstrap, Tailwind CSS, SCSS
  - **Backend:** Node.js, Laravel, Symfony
  - **Database:** MongoDB
  - **CMS & Tools:** WordPress, TypeScript
- Animated progress bars or hover effects on skill cards

### 5. Experience
- Vertical timeline layout
- Zignuts Technolab — Web Developer (2022–Present)
- Key responsibilities listed with bullet points

### 6. Projects
- Card-based grid with abstract gradient illustrations (no real screenshots)
- Generic project titles: CRM Platform, Sports Marketplace, Real Estate Platform, Education Platform
- Each card: description, responsibilities, tech stack badges
- Hover animations for engagement

### 7. Services
- Icon + card layout for 5 services:
  - UI/UX Design, Web Design, Frontend Dev, Full Stack Dev, Website Dev
- Subtle hover animations

### 8. Contact
- Contact form (Name, Email, Message) — client-side only, no backend
- Direct contact info: email, phone
- Social icons: LinkedIn, GitHub
- Clean minimal layout

---

## Design Details
- **Theme:** Light & Dark mode with smooth CSS transitions
- **Colors:** Primary blue, white, blue-gray, black accents with trendy gradients
- **Typography:** Modern sans-serif with bold editorial headings
- **Animations:** Scroll-reveal effects, hover transitions, smooth theme switching
- **Responsive:** Mobile-first, fully responsive across all breakpoints
- **Single page** with smooth scrolling between sections
